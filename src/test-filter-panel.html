Component Features

The FilterPanel component includes all required features from the AI_MATCHING_SPEC.md specification:

✅ Basic Filters

Make (multi-select dropdown), Model (dependent on make), Year Range (1990-2026), Price Range (10k-500k AED), Mileage (0-200k km)

✅ Advanced Filters

Body Type checkboxes, Fuel Type checkboxes, Transmission toggle, Color swatch picker (10 colors), Location multi-select

✅ Trust Filters

Accident-Free Only toggle, Service History Available toggle, Certified Pre-Owned toggle, Verified Seller toggle

✅ Filter Chips

Selected filters appear as removable chips above results with instant visual feedback

✅ Mobile Responsive

Collapses to drawer on small screens with mobile toggle button

✅ State Management

React Context API for global filter state management

✅ Dubai Theme

Blue/gold palette, Inter font, glass-morphism background, rounded corners, subtle shadows

✅ Event Emission

Emits change events to parent via onFilterChange prop
